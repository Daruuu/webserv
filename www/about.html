🚀 Que es webserv?

Webserv es un proyecto de la escuela 42 que consiste en implementar un servidor HTTP compatible con HTTP/1.1, similar a Nginx. Esta escrito completamente en C++98 sin usar librerias externas, utilizando I/O no bloqueante con epoll para manejar multiples conexiones de forma eficiente.

⚡ Funcionalidades

Metodos HTTP

Soporte completo para GET, POST y DELETE con manejo correcto de headers y body.

Archivos estaticos

Servir archivos HTML, CSS, JS, imagenes y mas desde el directorio raiz configurado.

CGI

Ejecucion de scripts Python y PHP a traves del protocolo Common Gateway Interface.

Upload de archivos

Subida de archivos al servidor mediante peticiones POST con multipart/form-data.

Virtual Hosts

Multiples servidores virtuales en diferentes puertos con configuraciones independientes.

Paginas de error

Paginas de error personalizadas (404, 500, etc.) configurables por servidor.

Autoindex

Listado automatico de directorios cuando no se encuentra un archivo indice.

Non-blocking I/O

Manejo eficiente de conexiones multiples usando epoll sin hilos adicionales.

🛠 Configuracion

El servidor se configura mediante archivos de configuracion similares a los de Nginx. Se pueden definir bloques server con directivas como listen, root, index, y bloques location para cada ruta.
